Nexora — UX Design Plans
4 distinct directions for improving the user experience. Each plan includes a desktop web view and a mobile web view.
Plan 01
Today-First Dashboard
Replace the static library home with a personalized daily action screen that puts due cards front-and-center.
✓ Keeps current sidebar ✓ Lowest effort to ship ~ Medium redesign Focus: Habit building
Key changes: The main page opens to a "Today" panel showing exactly how many items are due, with a big green CTA to start review. Quick-stats grid (streak, words mastered, accuracy) sits below. The daily queue badge moves from the sidebar into the main area where it gets more attention. The Library is demoted to a secondary tab.

Plan 02
Command-Center (Notion-Style)
Dense, information-rich workspace. Treat content like a database — filterable, sortable, keyboard-driven.
Power users Bigger sidebar ~ Medium redesign Focus: Data density
Key changes: Sidebar becomes a tree-nav (nested decks, collapsible). Main area shows a board view with metric cards at top + a scrollable table of all content. Per-row SM-2 state pills, due-date columns, deck tags. A small "Insights" panel on the right shows streak + mastery bar. Cmd+K is the primary way to jump anywhere.

Plan 03
Gamified Learning Path
Duolingo-meets-Brainscape. XP, streaks, skill nodes and achievement badges make studying feel like a game.
Habit-forming Most engaging High effort to ship Focus: Engagement
Key changes: XP bar and streak counter become primary status elements. Content is organized as "skill nodes" (like Duolingo's path) — tap a node to enter that deck's practice flow. Badges/achievements panel replaces the abstract stats view. Bottom nav always visible on all viewports. Green primary CTA for daily review is always above the fold.

Plan 04
Dark Editorial (Premium)
A deep dark-mode-first premium redesign. Sophisticated, high-contrast, Obsidian/Linear-inspired. Purple accent replaces emerald.
Premium feel Dark-first Highest effort Focus: Aesthetics
Key changes: Full dark mode as default (opt-in light mode). Purple/violet replaces green as primary accent — feels more "knowledge tool" than "flashcard app". Flashcard hero gets a gradient glow. Stats become beautiful charts. Typography uses larger, bolder weights for headings. The sidebar is slimmer (icon-only on smallest widths, expanding on hover).
